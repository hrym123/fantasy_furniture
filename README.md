# 幻想家具（Fantasy Furniture）

为 Minecraft 增添可用于装潢的**瓷砖**、**壁纸**与**屏风**，适合搭配建筑与室内设计。

---

## 运行环境

- **Minecraft**：1.20.1  
- **模组加载器**：Forge（请使用与当前 Minecraft 版本匹配的 Forge 安装包）

---

## 内容说明

### 瓷砖

共 **6 种颜色**：粉、黄、蓝、绿、青、紫。可作为地面或墙面贴面，质地偏硬，适合与石质、陶瓷类风格搭配。

- **建议工具**：使用**镐**挖掘效率更高。

### 壁纸

共 **6 种颜色**：粉、红、黄、蓝、绿、紫。质感偏软，适合营造柔和墙面效果。

- **建议工具**：使用**锄**挖掘效率更高。

### 屏风

一件较高的**装饰隔断**，可用来划分空间或作背景陈设。

- **占用空间**：屏风在竖直方向会占据**两格高度**，放置时请保证上方有足够空位，避免与天花板或相邻方块冲突。
- **建议工具**：使用**斧**挖掘效率更高。

---

## 小贴士

- 瓷砖、壁纸颜色较多，可与原木、羊毛、陶瓦等方块组合试搭。  
- 屏风适合放在房间转角、床头一侧或过道旁，作视觉分隔；若放置失败，先检查头顶是否有方块阻挡。
